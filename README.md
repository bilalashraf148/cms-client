### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

_To get started with the project._

1. Clone the repo
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the application
   ```sh
   npm start
   ```
